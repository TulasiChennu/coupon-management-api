
# Coupon Management API

Spring Boot REST API to manage coupons for an e-commerce platform.

## Implemented

- Coupon CRUD
- Cart wise coupon
- Applicable coupon calculation
- Strategy Pattern for extensibility

## Assumptions

- One coupon applied at a time
- Cart prices provided externally

## Future Improvements

- Product wise coupons
- BXGY coupons
- Coupon stacking
- Coupon usage tracking
- Redis caching
